# w-finance-react

[![NPM version](https://img.shields.io/npm/v/w-finance-react.svg?style=flat)](https://npmjs.org/package/w-finance-react)
[![NPM downloads](http://img.shields.io/npm/dm/w-finance-react.svg?style=flat)](https://npmjs.org/package/w-finance-react)

A react library developed with dumi

## 📦 安装


```bash
# npm
npm i w-finance-react -S

# pnpm
pnpm i w-finance-react -S 
```

## 🚀 快速开始

以下是 w-finance-react 的一些代码示例：

#### 代码片段:

```jsx
import ReactDOM from 'react-dom';
import { WFbankCardInput } from 'w-finance-react';

function App() {
  return <WFbankCardInput></WFbankCardInput>;
}

ReactDOM.render(<App />, mountNode);
```
